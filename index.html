Snapshot: How it works

  1. Capture NOAA APT FM audio from WebSDR.
  2. Demodulate (Hilbert / quadrature) & detect sync pulses.
  3. Rasterize audio lines into grayscale image channels.
  4. Enhance with Gaussian/Median filters; later: UNet/SRGAN.
  5. Provide downloads & analytics (NDVI, cloud %).
MVP tech stack:
Wix (frontend) + Python worker (FastAPI) for processing. Storage: S3 / signed URLs. AI: PyTorch for SR models (future).

Docs & Code Snippets

Signal → Audio

Use WebSDR to record NOAA APT FM audio (.wav). Example: 137.1 MHz. Libraries: GNU Radio, rtl_fm (if using local SDR).

Demodulation

Quadrature demod or Hilbert transform. In MATLAB: use hilbert() then unwrap phase or obtain instantaneous amplitude.

Rasterization

Detect sync pulses using cross-correlation; slice the audio stream into fixed-length lines and map amplitude to pixel intensity.

MATLAB snippet (demodulation + rasterization)

y = audioread('noaa.wav');
% approximate FM demod via Hilbert
z = hilbert(y);
inst_phase = unwrap(angle(z));
